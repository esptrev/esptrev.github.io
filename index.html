Development Projects

Clear To Close

Is a full-stack application that streamlines the home buying and selling process. The front-end is a user-friendly display that was composed using HTML, Tailwind CSS, Bootstrap JS and Jquery. Clear To Close uses a MySql database to track users and listings, along with an AWS S3 database for file upload.. Listing details are pulled through an outside API that is secured through Spring Boot. Users can register and login securly using Spring Security, OAuth 2.0 and JWT’s. The back-end is served with Spring Boot and custom API endpoints.The deployed project is hosted on an EC2 server and managed through GitHub Actions for CI/CD.

Movie Collection

Full stack project creating a personal movie database. Allowed users to query IMDb db through a RESTful API. Movies were then added to a SQL db allowing personal edits to ratings, plots and supporting actors/actresses. Program features an API developed using Java and MySQL. Front end utilized Javascript, Flex-box, Bootstrap, HTML, CSS, and jQuery to manipulate the DOM and its elements.
